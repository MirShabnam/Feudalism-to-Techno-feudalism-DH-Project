# Data README

This demo dataset compares **feudalism** and **technofeudalism** across concepts and events.

- `feud_techno_concepts.csv` — columns: `concept, feudalism_score, technofeudalism_score, note` (relative indicators)
- `feud_techno_events.csv` — columns: `year, era, label, type, intensity` (timeline markers)

Replace with your curated sources to scale the study.
