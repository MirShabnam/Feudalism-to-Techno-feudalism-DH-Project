# Feudalism → Technofeudalism — A Digital Humanities Project

**Theme:** A critical comparison between **medieval feudalism** and the rise of **modern technofeudal** power structures — examining how digital hierarchies may be re‑creating systems of control and servitude, revealing the subtle repetition of history in the age of technology.

_By Eprene_

## What this project does
- Builds small structured datasets comparing **feudal** vs **technofeudal** concepts and events
- Runs mixed‑method analysis: concept frequencies, score comparison, timeline, and a concept network
- Generates charts in `output/` and a Pages‑ready site in `docs/` with interpretation

## How to run (locally)
```bash
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
python code/analyze_feud_techno_project.py
```
Outputs will appear in `output/` and are embedded in `docs/index.md` (after you enable GitHub Pages).
